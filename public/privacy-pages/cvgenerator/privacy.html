CVGenerator

Politique de confidentialité

Dernière mise à jour : mai 2026

En résumé : CVGenerator traite votre profil et votre fiche de poste uniquement pour générer votre CV, via l'API Anthropic ou Google Vertex AI. Ces données ne sont jamais stockées sur nos serveurs. Seule votre clé API est conservée localement sur votre appareil.

1. Qui sommes-nous ?

CVGenerator est une application mobile Android de génération de CV assistée par intelligence artificielle, développée à titre personnel. Elle n'est pas exploitée par une société commerciale. Identifiant de l'application : com.stellasecret.cvgenerator.

2. Données traitées et finalité

CVGenerator traite uniquement les données que vous fournissez volontairement pour générer votre CV :

  • Votre profil : texte importé depuis un fichier (PDF, DOCX, TXT…) ou saisi manuellement. Ce contenu peut inclure votre nom, vos expériences professionnelles, vos compétences et votre formation.
  • La fiche de poste : texte importé ou collé, décrivant l'offre d'emploi ciblée. Ces données sont facultatives.

Ces données sont transmises une seule fois, à la demande explicite de l'utilisateur (appui sur « Générer mon CV »), à l'API d'IA choisie, puis immédiatement écartées. Elles ne sont jamais enregistrées sur un serveur exploité par CVGenerator.

3. Services d'IA tiers utilisés

Selon le mode choisi, CVGenerator transmet vos données à l'un des services suivants :

  • API Anthropic (Claude) : utilisée lorsque vous saisissez votre propre clé API ou pour les comptes non-premium. La transmission est soumise à la politique de confidentialité d'Anthropic. Anthropic indique ne pas utiliser les données des requêtes API pour entraîner ses modèles.
  • Google Vertex AI : utilisé uniquement pour les comptes premium connectés via Google. La transmission est soumise à la politique de confidentialité de Google Cloud.

⚠ La gestion des données transmises à ces services est régie par leurs propres politiques, indépendamment de CVGenerator. Nous vous recommandons de ne pas inclure d'informations sensibles (numéro de sécurité sociale, données bancaires, données de santé) dans votre profil.

4. Données stockées localement sur votre appareil

CVGenerator stocke les données suivantes uniquement sur votre appareil, via Android DataStore (stockage chiffré par Android) :

  • Votre clé API Anthropic (si vous en saisissez une) — stockée localement, jamais transmise à nos serveurs.

✓ Aucun historique de CV généré n'est conservé.
✓ Votre profil et votre fiche de poste ne sont jamais sauvegardés entre les sessions.
✓ La désinstallation efface l'intégralité des données locales.

5. Authentification Google (optionnelle)

CVGenerator propose une connexion via votre compte Google (OAuth 2.0 via Firebase Authentication). Cette fonctionnalité est entièrement optionnelle.

Lorsque vous vous connectez :

  • Votre adresse e-mail Google est utilisée pour identifier votre session et vérifier si votre compte bénéficie d'un accès premium (Vertex AI).
  • Aucun accès à vos emails, contacts ou autres données Google n'est demandé.
  • Vous pouvez vous déconnecter à tout moment depuis l'application.
  • Vous pouvez révoquer l'accès depuis Google Account → Applications tierces.

La gestion de l'authentification est soumise à la politique de confidentialité de Google.

6. Données non collectées

CVGenerator ne collecte, ne stocke et ne transmet aucune des données suivantes à nos serveurs :

✕ Données de localisation
✕ Identifiants publicitaires
✕ Données de navigation ou d'usage
✕ Données biométriques ou de santé
✕ Contacts ou données d'autres applications
✕ CV générés ou historique des générations

7. Publicité et analytics

CVGenerator ne contient aucune publicité et n'intègre aucun outil d'analyse ou de suivi comportemental (pas de Firebase Analytics, pas de Crashlytics, pas de SDK publicitaire, pas de pixel de tracking).

✓ Aucune publicité.
✓ Aucun SDK de tracking.
✓ Aucune donnée vendue à des tiers.

8. Permissions Android utilisées

  • INTERNET — nécessaire pour appeler l'API d'IA (Anthropic ou Vertex AI) et pour l'authentification Google.
  • READ_EXTERNAL_STORAGE / READ_MEDIA_* — nécessaire pour lire les fichiers (PDF, DOCX, TXT…) que vous choisissez d'importer depuis votre appareil. L'accès n'est déclenché que lors de la sélection d'un fichier par l'utilisateur.

CVGenerator ne demande aucune permission de localisation, d'appareil photo, de contacts, de microphone ou d'accès aux SMS.

9. Sécurité

Les communications avec les API d'IA sont chiffrées via HTTPS/TLS. Votre clé API est stockée dans Android DataStore, dont le contenu est protégé par le système de chiffrement Android.

Nous vous recommandons de ne pas partager votre appareil déverrouillé avec des tiers si vous avez enregistré votre clé API dans l'application.

10. Vos droits (RGPD)

CVGenerator ne traitant aucune donnée personnelle sur ses propres serveurs, les droits d'accès, de rectification et de suppression s'exercent directement sur votre appareil ou auprès des services tiers concernés.

  • Suppression locale : désinstallez l'application ou supprimez votre clé API depuis les paramètres.
  • Données transmises aux API : contactez directement Anthropic ou Google Cloud pour exercer vos droits relatifs aux données transmises à leurs services.
  • Authentification Google : révoquez l'accès depuis myaccount.google.com/permissions.

11. Enfants

CVGenerator est destiné aux personnes en recherche d'emploi et n'est pas conçu pour les enfants de moins de 13 ans. Nous ne collectons sciemment aucune information concernant des mineurs.

12. Modifications de cette politique

En cas de modification de cette politique, la date de mise à jour sera modifiée en haut de cette page. Les changements significatifs seront signalés dans les notes de mise à jour de l'application sur le Google Play Store.

13. Contact

Pour toute question concernant cette politique de confidentialité : contact@stellasecret.com
